# Manipulación de Archivos y Buffers en NvChad (Neovim)

Aquí tienes una tabla con los comandos esenciales para manipular archivos y buffers. Recuerda que un **buffer** es la representación en memoria de un archivo abierto (o un archivo nuevo, sin guardar), y una **ventana** es una vista de un buffer.

| Comando         | Descripción                                                                 | Notas Importantes                                                                                                                                                                                                                                                                                                                               |
| :-------------- | :-------------------------------------------------------------------------- | :---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Abrir** |                                                                             |                                                                                                                                                                                                                                                                                                                                                                 |
| `:e <ruta>`     | Abre un archivo en la ventana actual. Si el archivo no existe, lo crea.    | Si ya tienes cambios sin guardar, te pedirá guardar o usar `!`.                                                                                                                                                                                                                                                                                 |
| `:sp <ruta>`    | Abre un archivo en una nueva ventana dividida horizontalmente.              | Ideal para ver dos archivos al mismo tiempo.                                                                                                                                                                                                                                                                                                    |
| `:vsp <ruta>`   | Abre un archivo en una nueva ventana dividida verticalmente.                | También muy útil para comparar o copiar entre archivos.                                                                                                                                                                                                                                                                                         |
| `:tabe <ruta>`  | Abre un archivo en una nueva pestaña (tab).                                 | Útil para organizar grupos de archivos relacionados. Puedes tener varias ventanas en cada pestaña.                                                                                                                                                                                                                                               |
| **Cerrar** |                                                                             |                                                                                                                                                                                                                                                                                                                                                                 |
| **`:bd`** | **Cierra el buffer actual** (lo elimina de la lista de buffers cargados).   | **Este es el comando clave que buscabas.** Cierra el archivo actual sin cerrar Neovim, incluso si es el único buffer visible. Si el buffer tiene cambios, te pedirá guardar o usar `!`. Si es el único buffer y ventana, cerrará Neovim.                                                                                              |
| `:bd!`          | Cierra el buffer actual **sin guardar** los cambios.                        | ¡Úsalo con precaución!                                                                                                                                                                                                                                                                                                                          |
| `:q`            | Cierra la ventana actual. Si es la única ventana, cierra Neovim.           | Si el buffer en esa ventana tiene cambios sin guardar, te pedirá guardar o usar `!`.                                                                                                                                                                                                                                                           |
| `:q!`           | Cierra la ventana actual **sin guardar** los cambios. Si es la única ventana, cierra Neovim sin guardar. |                                                                                                                                                                                                                                                                                                                                                                 |
| `:wq`           | Guarda y cierra la ventana actual. Si es la única ventana, guarda y cierra Neovim. |                                                                                                                                                                                                                                                                                                                                                                 |
| `:wqa`          | Guarda y cierra **todos los buffers** y sale de Neovim.                     | Equivalente a `:wa` (guardar todo) y luego `:qa` (cerrar todo).                                                                                                                                                                                                                                                                                  |
| `:tabc`         | Cierra la pestaña actual.                                                   | Cierra todas las ventanas dentro de esa pestaña.                                                                                                                                                                                                                                                                                                |
| `:only`         | Cierra **todas las demás ventanas**, dejando solo la ventana actual.        | Útil cuando tienes muchas divisiones y quieres concentrarte en un solo archivo.                                                                                                                                                                                                                                                                 |
| **Navegar** |                                                                             |                                                                                                                                                                                                                                                                                                                                                                 |
| `:ls` o `:buffers` | Muestra una lista de todos los buffers cargados.                          | Los buffers con `%a` son activos, `#` son alternos. `h` indica un buffer oculto.                                                                                                                                                                                                                                                                |
| `:b <número_buffer>` | Salta a un buffer específico por su número (lo ves con `:ls`).             | Por ejemplo, `:b 3` para ir al buffer número 3.                                                                                                                                                                                                                                                                                                 |
| `:b <nombre_buffer>` | Salta a un buffer específico por su nombre (o parte de él).                 | Puedes usar `Tab` para autocompletar.                                                                                                                                                                                                                                                                                                   |
| `:bp`           | Salta al **buffer previo** en la lista.                                     |                                                                                                                                                                                                                                                                                                                                                                 |
| `:bn`           | Salta al **buffer siguiente** en la lista.                                  |                                                                                                                                                                                                                                                                                                                                                                 |
| `<C-^>` (Ctrl+^) | Alterna rápidamente entre el buffer actual y el último buffer visitado.    | Muy útil para cambiar rápidamente entre dos archivos.                                                                                                                                                                                                                                                                                           |
| `<C-w>w`        | Salta a la siguiente ventana dividida.                                      | Rota entre las ventanas de tu sesión.                                                                                                                                                                                                                                                                                                           |
| `<C-w>h`/`<C-w>j`/`<C-w>k`/`<C-w>l` | Salta a la ventana a la izquierda/abajo/arriba/derecha. | Muy intuitivo para moverse entre splits.                                                                                                                                                                                                                                                                                                        |
| `gt`            | Salta a la siguiente pestaña.                                               |                                                                                                                                                                                                                                                                                                                                                                 |
| `gT`            | Salta a la pestaña previa.                                                  |                                                                                                                                                                                                                                                                                                                                                                 |
