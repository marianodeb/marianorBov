## Python Built in Functions

---

Python has a set of built-in functions.

| Función | Descripción |
| --- | --- |
| abs()| Devuelve el valor absoluto de un número |
| all()| Devuelve True si todos los elementos en un objeto iterable son verdaderos |
| any()| Devuelve True si algún elemento en un objeto iterable es verdadero |
| ascii()| Devuelve una versión legible de un objeto. Reemplaza caracteres no ASCII con caracteres de escape |
| bin()| Devuelve la versión binaria de un número |
| bool()| Devuelve el valor booleano del objeto especificado |
| bytearray()| Devuelve un arreglo de bytes |
| bytes()| Devuelve un objeto bytes |
| callable()| Devuelve True si el objeto especificado es invocable, de lo contrario False |
| chr()| Devuelve un carácter del código Unicode especificado |
| lassmethod() | Convierte un método en un método de clase |
| compile()| Devuelve el código fuente especificado como objeto, listo para ejecutarse |
| complex()| Devuelve un número complejo |
| delattr()| Elimina el atributo especificado (propiedad o método) de un objeto |
| dict()| Devuelve un diccionario (array) |
| dir()| Devuelve una lista de propiedades y métodos del objeto especificado |
| divmod()| Devuelve el cociente y el resto cuando el argumento1 se divide por el argumento2 |
| enumerate()| Toma una colección (ej. una tupla) y la devuelve como objeto enumerado |
| eval()| Evalúa y ejecuta una expresión |
| exec()| Ejecuta el código (u objeto) especificado |
| filter()| Usa una función filtro para excluir elementos en un objeto iterable |
| float()| Devuelve un número de punto flotante |
| format()| Da formato a un valor especificado |
| frozenset()| Devuelve un objeto frozenset |
| getattr()| Devuelve el valor del atributo especificado (propiedad o método) |
| globals() | Devuelve la tabla de símbolos global actual como diccionario |
| hasattr()| Devuelve True si el objeto especificado tiene el atributo (propiedad/método) indicado |
| ash()| Devuelve el valor hash de un objeto especificado |
| elp()| Ejecuta el sistema de ayuda integrado |
| hex()| Convierte un número a hexadecimal |
| id()| Devuelve el identificador de un objeto |
| input()| Permite la entrada del usuario |
| int()| Devuelve un número entero |
| isinstance()| Devuelve True si un objeto es instancia del objeto especificado |
| issubclass()| Devuelve True si una clase especificada es subclase de otro objeto |
| iter()| Devuelve un objeto iterador |
| len()| Devuelve la longitud de un objeto |
| list() | Devuelve una lista |
| locals()| Devuelve un diccionario actualizado de la tabla de símbolos local |
| map() | Devuelve el iterador especificado con la función aplicada a cada elemento |
| max()| Devuelve el elemento más grande de un iterable |
| memoryview()| Devuelve un objeto memoryview |
| min() | Devuelve el elemento más pequeño de un iterable |
| next() | Devuelve el siguiente elemento en un iterable |
| object() | Devuelve un nuevo objeto |
| oct()| Convierte un número a octal |
| open()| Abre un archivo y devuelve un objeto archivo |
| ord()| Convierte un carácter a su entero Unicode correspondiente |
| pow() | Devuelve el valor de x elevado a y |
| print() | Imprime en el dispositivo de salida estándar |
| roperty() | Obtiene, establece o elimina una propiedad |
| range()| Devuelve una secuencia numérica empezando en 0 (por defecto) |
| epr() | Devuelve una versión legible de un objeto |
| reversed()| Devuelve un iterador invertido |
| round()| Redondea números |
| set()| Devuelve un nuevo objeto conjunto |
| setattr() | Establece un atributo (propiedad/método) de un objeto |
| slice() | Devuelve un objeto de segmento |
| sorted() | Devuelve una lista ordenada |
| staticmethod() | Convierte un método en estático |
| str() | Devuelve un objeto cadena |
| sum() | Suma los elementos de un iterador |
| super()| Devuelve un objeto que representa la clase padre |
| tuple()| Devuelve una tupla |
| type()| Devuelve el tipo de un objeto |
| vars()| Devuelve la propiedad `\_\_dict\_\_` de un objeto |
| zip()| Devuelve un iterador a partir de dos o más iteradores |
